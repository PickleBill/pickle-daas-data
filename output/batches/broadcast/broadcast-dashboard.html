Add real ElevenLabs voices — Ron Burgundy, ESPN anchor, Chuck Norris energy.
1. Get your API key at elevenlabs.io → Profile → API Keys
2. Add it to your .env file: ELEVENLABS_API_KEY=your_key
3. Run: python elevenlabs-voice-pipeline.py --analysis output/broadcast/analysis_*.json --voice ron_burgundy
4. The script generates an MP3 per clip → drop into this player or merge with the video via FFmpeg.
